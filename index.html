GuitarLog 개인정보처리방침

시행일: 2026년 3월 28일 | 최종 수정: 2026년 4월 8일

1. 수집하는 개인정보

GuitarLog(이하 "앱")은 클라우드 백업 기능 사용 시 다음 정보를 수집합니다.

Google 로그인 정보: 이름, 이메일 주소, 프로필 사진 (Google Sign-In을 통해 제공)

연습 데이터: 사용자가 직접 입력한 곡 정보 및 연습 기록 (클라우드 백업 선택 시에만 서버에 저장)

Google 로그인을 사용하지 않는 경우, 모든 데이터는 기기 내에만 저장되며 외부로 전송되지 않습니다.

2. 개인정보 이용 목적

수집된 정보는 다음 목적으로만 사용됩니다.

- 사용자 인증 및 계정 식별

- 클라우드 백업 및 기기 간 데이터 동기화

- 앱 서비스 제공 및 개선

3. 개인정보 보관 및 삭제

기기 내 데이터: 앱 삭제 시 자동으로 삭제됩니다.

클라우드 데이터: Google Firebase에 저장되며, 계정 삭제 요청 시 즉시 삭제됩니다. 별도 보관 기간 없이 즉시 영구 삭제됩니다.

4. 계정 및 데이터 삭제 방법

GuitarLog 앱에서 직접 계정을 삭제할 수 있습니다.

삭제 절차:

1. 앱 실행 후 하단 탭에서 설정으로 이동

2. 계정 섹션에서 Google 계정으로 로그인되어 있는지 확인

3. 하단 위험 구역 섹션의 계정 삭제 버튼 탭

4. 확인 다이얼로그에서 삭제 선택

5. Google 계정 재인증 후 계정 및 모든 클라우드 데이터가 즉시 영구 삭제됩니다

삭제되는 데이터: Firebase 인증 계정, Firestore에 저장된 모든 연습 기록 및 곡 정보

삭제 후 보관 기간: 없음 (즉시 영구 삭제)

이메일로 삭제 요청을 원하시는 경우: guitarlog.dev@gmail.com으로 문의해 주세요.

5. 제3자 제공

앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음 서비스를 이용합니다.

Google Firebase: 인증 및 클라우드 데이터 저장 (Firebase 개인정보처리방침)

6. 아동 개인정보 보호

앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 이용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

- Google 로그인 해제 (앱 내 설정 → 로그아웃)

- 계정 및 모든 클라우드 데이터 삭제 (앱 내 설정 → 위험 구역 → 계정 삭제)

- 앱 삭제를 통한 모든 로컬 데이터 삭제
